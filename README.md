# Sobre el proyecto
Expreso a Oriente.com es un vlog de 4 amigos que viajaron en el año 2012 y 2013 por Europa, Asia y Medio Oriente. Documentaron todo su viaje en este sitio.

El sitio fue originalmente hecho en Wordpress pero fue hackeado.

La cantidad de URLS es limitadas, por lo que se clonarà el sitio con Claude Code.
